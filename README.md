Sneaker maison
Đây là một concept website dành cho thương hiệu giày cao cấp (giả tưởng). Lấy cảm hứng từ sự tối giản và cấu trúc hình khối mạnh mẽ, trang web tập trung vào việc loại bỏ các chi tiết thừa để tôn vinh sản phẩm thông qua khoảng trắng và độ tương phản cao.
